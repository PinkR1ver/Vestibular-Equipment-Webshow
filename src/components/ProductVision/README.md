# ProductVision 组件

## 新定位
这一段不再做传统的三张 feature card，而是改成“产品家族 / 系统矩阵页”。

目标：
- 与上方 Market Reality 的竞品展陈形成明显区分
- 用更产品化、更系统化的方式展示我们的解决方案
- 在最终 hand-drawn / Gemini 产品图到位前，先用稳定的 UI 骨架承接产品想法

## 页面结构
- 左侧：三个产品卡
  - 居家检测眼动设备
  - 居家恢复训练设备
  - 诊间快速评估设备
- 右侧：共享智能能力中枢
  - Eye-motion capture
  - AI interpretation
  - Training feedback
  - Cloud case sync

## 设计原则
1. 视觉语气从“市场痛点/传统设备”切换到“未来产品体系”
2. 减少红色警示感，增加 cyan / violet / emerald 的解决方案气质
3. 为后续产品图预留固定的 render slot，避免生成图片前频繁改结构
4. 让用户感知这不是三台孤立设备，而是一套贯穿家庭与临床的连续产品系统
