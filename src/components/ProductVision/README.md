# ProductVision 组件

## 作用
页面终章（Section 4）。基于前面抛出的所有问题（昂贵、难用、缺口），给出我们的“降维打击”方案：便宜硬件+AI辅助+居家化。

## 动画与交互设计
1. **背景变调**：该区块使用带有 Emerald（翠绿色）渐变的背景色和巨大光晕（`blur-[100px]`），从整体的暗色系中脱颖而出，代表着“生机”与“解决之道”。
2. **苹果风大字报**：使用居中且极简的大标题“重塑前庭评估”。`h2` 和副标题分开带有淡入上浮效果。
3. **弹性视效**：下方三个核心功能特性使用具有 `bounce: 0.4` 的阻尼弹性动画，增强科技感和灵动感。
