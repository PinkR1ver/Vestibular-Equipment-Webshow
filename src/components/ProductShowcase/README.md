# ProductShowcase

## 定位
这是 Our Product 的章节式主展示区，不再做三栏并列展台，而改成“像书一样逐页展开”的产品章节。

## 页面结构
1. 引导页：说明这是一个由三类终端构成的产品家族
2. Chapter 01：居家检测眼动设备
3. Chapter 02：居家恢复训练设备
4. Chapter 03：诊间快速评估设备

## 设计原则
- 每个产品单独占据一页，避免并列展台的信息竞争
- 视觉上保留“堆叠书页 / 章节页”的感觉
- 向下滚动时完成背景色、文案和主视觉切换
- 第一张产品页优先接入已有手绘草图，后续章节继续等待 hand-drawn / Gemini 图
