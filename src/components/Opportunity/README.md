# Opportunity 组件

## 作用
对应 PRD 中 Section 3，向投资人/受众展示“钱在哪里”。着重强调由于技术革新（从笨重到便携）所释放出来的原本不存在的蓝海市场规模。

## 动画与交互设计
1. **数字震撼**：顶部展示了夸张的 50-100亿估算大数字，给足预期。
2. **卡片缩放弹射**：下方的三个细分市场卡片使用了 `scale: 0.8` -> `scale: 1` 配合 `spring` 阻尼特效的弹出动画，让数据的展示更具生命力。
3. **Hover 特效**：使用了 Tailwind 的 `group-hover` 控制卡片右上角的绝对定位光晕（blur）放大，增加鼠标经过时的现代极客化交互反馈。
